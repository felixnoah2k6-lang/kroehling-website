# Website UI Kit — Schönhärl · Kroehling · Gloßner

A single-page hi-fi recreation of the redesigned practice website.

## Files
- `index.html` — full page, mounts all components.
- `Header.jsx` — sticky top bar with logo, nav, click-to-call.
- `Hero.jsx` — display headline, opening hours, phone CTA, hero photo.
- `Philosophy.jsx` — "Wir behandeln Menschen, keine Symptome".
- `Team.jsx` — three doctors + MFA grid.
- `Services.jsx` — Leistungen tiles (sage icons).
- `Locations.jsx` — Pielenhofen + Wolfsegg cards with hours.
- `News.jsx` — Aktuelles list.
- `Contact.jsx` — phone/fax/email + accessibility.
- `Footer.jsx` — Impressum, Datenschutz, copyright.

## Notes
- All copy in German, "Sie" form throughout.
- Email shown only in Footer with the "keine Patientenanfragen" caveat.
- No online booking — phone is the only CTA.
- Lucide icons via CDN.
