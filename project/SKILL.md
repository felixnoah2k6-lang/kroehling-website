Brand essentials:
- All copy in German, formal "Sie" form, never "Du", no emoji.
- Phone always rendered as `09409 / 86290` (with spaced slash).
- E-mail only in Impressum, with the "keine Patientenanfragen" caveat.
- Primary palette: warm cream `#faf7f1` base + Naabtal sage `#5e8556` primary + warm ink `#2a2620` text. Logo orange `#f0a020` reserved for cautionary notes.
- Type pairing: Source Serif 4 for display / headings, Source Sans 3 at 17px body.
- Use real photography (assets/), Lucide icons stroke 1.75, soft 8–24px radii, warm-grey shadows. Never pure white, no blue, no gradients.
